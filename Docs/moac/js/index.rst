=================================
Chain3 JavaScript version 1.0.x 
=================================

chain3.js 1.0.x is developed based on web3 1.2.1 JavaScript library to communicate with MOAC network. 
It is still under development. Users can submit the issues to `Chain3 JS lib <https://github.com/MOACChain/chain3/issues>`_.

The following documentation will guide you through :ref:`installing and running chain3.js <adding-chain3>`,
as well as providing a :ref:`API reference documentation <api-ref>` with examples.
