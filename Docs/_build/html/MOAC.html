# 欢迎来到墨客的世界！

MOAC旨在提供一种可扩展且有弹性的区块链，支持基于分层结构的状态交易，数据访问，和控制流程。它创建了一个框架以允许用户用高效的方式执行智能合约。它还提供了开发的体系结构，采用底层基础设施来快速简便地产生子区块链。它是一个区块链平台，可以为子区块链的架设提供必要的部件，为想法测试，私链部署，复杂任务处理和智能合同应用等提供解决方案。
